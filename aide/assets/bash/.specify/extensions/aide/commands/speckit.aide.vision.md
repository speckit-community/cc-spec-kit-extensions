# speckit.aide.vision

Create or refine the project vision document for AI-driven development.

## Usage

Invoke this command to establish or update the project's high-level vision.

## Behavior

1. Check if a vision document exists in `.specify/memory/vision.md`
2. If not, create a new vision document from user input
3. If exists, present current vision and allow refinement
4. Validate vision covers: problem statement, target users, success metrics, scope boundaries
5. Save updated vision document
